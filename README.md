这是一个简单的脚手架工具，主要实现初始化Koa模板，

# 使用方式

安装CLI工具为全局

```shell
npm i wuyan_koacli -g
```

初始化：

```shell
WY init koa paoject-name
```

# 二次开发

修改`template.json`文件

```json
{
  "koa": "xiaopalu/koa_template"
}
```

`koa`是模板名称，`"xiaopalu/koa_template"`是模板仓库地址
